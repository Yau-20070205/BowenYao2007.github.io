# Yau Blog

Personal blog for Yau (姚博文), hosted at https://yau-20070205.github.io/.

## Development

1. Install Ruby and Bundler.
2. Install dependencies:
   ```sh
   bundle install
   ```
3. Serve the site:
   ```sh
   bundle exec jekyll serve
   ```

## Assets

If you need to rebuild CSS/JS assets, use Grunt:

```sh
grunt watch
```
